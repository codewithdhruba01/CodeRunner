# Welcome to CodeRunner!

A fully functional online code editor and compiler built with Next.js 13.

## What is CodeRunner?

CodeRunner is a modern, responsive web application that allows users to:
- Write code in **C, C++, Java, and Python**
- **Compile and execute** code instantly
- **Share code** with others via unique links
- Switch between **dark and light themes**
- Use on **any device** (mobile, tablet, desktop)

## Fastest Way to Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Set up Supabase** (2 minutes)
   - Create free account at [supabase.com](https://supabase.com)
   - Create new project
   - Copy URL and key to `.env.local`

3. **Run the app**
   ```bash
   npm run dev
   ```

4. **Open browser**
   - Visit http://localhost:3000
   - Start coding!

Full instructions: [QUICKSTART.md](QUICKSTART.md)
